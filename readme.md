# Inicializando o serviço do mysql pelo docker compose
 Requisitos: 

 - Instalação do docker compose e docker.

Iniciar o container

``` $ docker compose up -d ```

Entrar no container localmente

``` $ docker exec -it mk-mysql bash```

Entrar no banco de dados

``` $ mysql -u root -p```
